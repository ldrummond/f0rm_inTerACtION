Recursive Iframes

Project to explore more. Created a system for iFrame recursion. Building on Jonathan's mention of a nested google street view project, I was wondering if it was possible to point an iframe at itself. Succeeded through using query parameters (different urls for each iframe, same content). Result lets you recursively nest whatever you want!
